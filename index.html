Git things to remember

You need to first give the Name and email to the git For initialzation

    existing
  • git config --global user.name "Cherry"
  • git config --global user.email "saicharanbk2006@gmail.com"

Then for creation you need to do these steps in the folder you need to upload

  • git init
  • git add README.md (Optional)
  • git commit -m "your comment"
  • git add .
  • git status (All files should be in green then you are ready to go)
  • git remote add origin (Your Repo link, which ends with .git)
  • git push -u origin master (or) git push origin master (Where master the branch which you have selected)

If you want to update the existing folder to the git the do these steps

  1. git add .
  2. git commit -m "Message"
  3. git pull origin master (If the repo is used and saved by someone the you need to do this)
  4. git push origin master (This makes the changes that you have done in the local storage)

To ignore a certian file

  • Create a file called .gitignore
  • Enter the file name with extention and the file will be excluded
  • to ignore all files of same type then " *.(filetype) "
  • Note you should create in the git folder

Additional commands

  • git reset (This will reset the files and types)
